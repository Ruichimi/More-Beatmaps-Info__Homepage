Privacy Policy

Last updated: December 2025

What data we collect

More Beatmaps Info collects only the client’s IP address and monitors the number of requests made to its server for security and service stability purposes.

IP addresses are stored temporarily in server memory (RAM) and are automatically deleted upon server restart. They are not written to disk and are not permanently stored.

The only exception applies to IP addresses exhibiting highly suspicious or abusive activity. Such IP addresses may be securely stored in a database solely for the purpose of enforcing permanent access restrictions.

Access to this information is strictly limited to the server owner, who is a single individual. Collected data is never shared with, sold to, or disclosed to any third parties.

Browser permissions

The extension only accesses https://osu.ppy.sh related pages in order to analyze beatmaps information and display additional information about them.

Caching

Some beatmaps information is partially stored in the browser’s local storage for fast display without making a request to the server.

The amount of stored data is limited and is automatically cleared when a certain low limit is reached. It does not take up any noticeable disk space.

Contact

If you have any privacy or other extension-related questions, you can contact the developer via this page or GitHub.
